# Schattenlaufen 🏃‍♂️🌑  

Ein interaktives HTML-Story-Spiel über das Verfolgen eines geheimnisvollen Schattens.  

## 📌 Features
- 📖 Wähle deinen eigenen Weg
- 🌆 Atmosphärische Bilder
- 🔗 Interaktive Links
